## General info

> This is a project of an application where users can "share" car seats and travel together. The inspiration for this app was a BlablaCar application.


## Technologies
* Java
* Spring (Data,Boot,Security)
* Hibernate
* MySQL
* JUNIT
* Lombok
* BootStrap
* Thymeleaf
* HTML, CSS, JS

## Features
